# ChatGPT Prompt Navigator

ChatGPT Prompt Navigator is a lightweight Chrome extension that adds a prompt navigation panel to ChatGPT. It allows you to quickly view and jump to any of your previous prompts in the current conversation.

The extension works entirely inside your browser and enhances navigation without modifying ChatGPT's functionality.

---

# Features

* Adds a " 🕒History " button to the ChatGPT header
* Displays a panel listing all your prompts
* Click any prompt to instantly scroll to it
* Automatically works when switching between chats
* Closes when clicking outside the panel

---

# Privacy

This extension is completely client-side.

* No data is sent anywhere
* No tracking
* No analytics
* No storage of conversations
* No external network requests

All processing happens locally in your browser.

---

# Demo Behaviour

1. Open ChatGPT
2. Click the 🕒History button in the header
3. A panel opens showing your prompts
4. Click any prompt to navigate to it
5. Click outside the panel to close it
6. Switching chats keeps the extension active automatically

Note- If the button does not appear, try clicking on the icon in extensions toolbar

---

# Installation

1. Download or clone this repository

2. Open Chrome and go to: chrome://extensions

3. Enable Developer mode

4. Click Load unpacked

5. Select the extension folder

6. Open https://chatgpt.com

The 🕒History button will appear in the header.
